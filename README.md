# East Coast USA Trip Planner 🗺️

  An interactive 14-day East Coast USA trip planner built with React + Vite.

  **Route:** NYC → Washington DC → Philadelphia → Boston  
  **Dates:** June 20 – July 4, 2026  
  **Traveler:** Solo Indian vegetarian from Mumbai

  ## Features
  - 📅 14-day day-by-day itinerary with morning/afternoon/evening plans
  - 🗺️ Interactive Leaflet map with all stops
  - 🏨 Hotels for all 4 cities with booking links
  - 🍽️ Vegetarian food guide (Indian, global cuisines)
  - 🛫 Flight options with booking links
  - 💰 Budget tracker
  - ☕ Specialty coffee shops
  - 🏃 Running routes
  - 🥾 Hiking & day trips
  - 💎 Hidden gems
  - 🎵 Music & nightlife guide
  - 🎬 Bollywood filming locations tour (NYC)

  ## Tech Stack
  - React + Vite + TypeScript
  - Tailwind CSS + shadcn/ui
  - Leaflet / React-Leaflet for maps
  - pnpm monorepo

  ## Getting Started
  ```bash
  pnpm install
  pnpm --filter @workspace/east-coast-trip run dev
  ```
